# [puter-park](https://puter.us)

업무/개인적으로 필요한 도구 등을 구현한 공원

### [tools]
1. 접속 IP 확인
2. 주간보고 변환
3. 월간보고 변환
4. 문자 인코딩/디코딩
5. 파일 크기 변환
6. unixtime 변환
7. shorten url
8. qr code 생성기
9. 문장 감성분석기 (with. [lucaseo](https://github.com/lucaseo))
10. json beautify/minify
11. progress (날짜 기반 진척률 확인)

---
* `openjdk 11`  
* `spring boot 2.7`  
* `jpa`  
* `javascript`  
* `mariadb 10.2`  
* `embed tomcat 9`  
* `redis 6`
